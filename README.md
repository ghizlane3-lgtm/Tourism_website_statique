# Tourism_website_statique

BOUKITOUR – Tourism Website
📌 Overview

BOUKITOUR is a responsive tourism web application developed to promote and showcase some of Morocco’s most remarkable travel destinations. The project provides users with an interactive platform to explore cities, discover activities, browse accommodations, and view local attractions.

The website focuses on delivering an organized and user-friendly experience for travelers interested in exploring Morocco.

✨ Features
Responsive Design
Optimized for desktop, tablet, and mobile devices.
Multi-Page Architecture
Structured navigation between multiple pages for different destinations.
Destination Exploration
Dedicated pages presenting information about several Moroccan destinations including:
Chefchaouen
Sahara Desert
Ifrane
Akchour
Detailed City Sections
Each destination includes:
Activities & Attractions
Hotels & Accommodation
Restaurants
Image Gallery
Interactive User Interface
Dynamic elements implemented with JavaScript to improve navigation and user experience.
🛠️ Technologies Used
HTML5 → Structure and content organization
CSS3 → Styling, layout, and responsive design
JavaScript → Interactivity and dynamic behaviors
🎯 Project Objectives

The goal of this project was to design and develop a tourism platform capable of presenting travel-related information in a structured and visually appealing way while applying front-end development principles.

Main objectives included:

Building a fully responsive interface
Designing a scalable multi-page website architecture
Improving front-end development skills
Creating an intuitive user experience for visitors
📚 Skills Developed

Through this project, the following skills were strengthened:

Front-End Web Development
Responsive Web Design
UI Structuring and Layout Design
JavaScript DOM Manipulation
Website Navigation Architecture
